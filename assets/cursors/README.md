# Cursor Assets

This folder should contain cursor images for the OBS Zoom Pro cursor overlay feature.

## Required Files

Place the following PNG files in this folder:

1. **arrow.png** - Default mouse arrow cursor (32x32 recommended)
2. **pointer.png** - Hand/pointer cursor for links (32x32 recommended)
3. **ibeam.png** - Text insertion cursor (I-beam) (32x32 recommended)

## Notes

- Images should be 32x32 pixels for best results
- PNG format with transparency (alpha channel)
- The script will automatically use these images for the cursor overlay
- If files are missing, the cursor overlay will be disabled

## Obtaining Cursor Images

You can use any cursor image editor or download cursor packs. Make sure to:
- Use PNG format
- Include transparency (alpha channel)
- Keep the size around 32x32 pixels
